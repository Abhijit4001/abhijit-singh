Designed and built an AI-based attendance system that automates student attendance tracking through facial recognition.

- Implemented facial detection and recognition using OpenCV and Python.
- Integrated MySQL database for storing and retrieving student records.
- Built a Tkinter-based user interface for easy operation by staff.
- Applied real-time video stream processing for live recognition.
