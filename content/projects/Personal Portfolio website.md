Built a full-stack personal portfolio website to showcase projects, blog posts, and professional experience with a polished, animated UI.

- Implemented server-side rendering with TanStack Start for fast initial page loads and SEO.
- Used Content Collections for type-safe, schema-validated markdown content management.
- Designed animated UI with TailwindCSS v4: typewriter hero text, floating gradient blobs, scroll-reveal transitions, and glassmorphism-style cards.
- Deployed on Netlify with automatic form handling, CDN edge delivery, and CI/CD on every push.
