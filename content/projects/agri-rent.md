Developed a full-stack platform that connects farmers with agricultural equipment rentals, powered by a data-driven backend.

- Designed backend APIs using FastAPI following RESTful principles.
- Implemented a Streamlit-based frontend dashboard for browsing and renting equipment.
- Used MongoDB for storing rental listings, user data, and transaction records.
- Applied data validation using Pydantic models to ensure data integrity.
